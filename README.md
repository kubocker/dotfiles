# minimum dotfiles

### set up
```bash
$ curl -L -O https://github.com/kubocker/dotfiles/tarball/master
$ tar xvf master
$ mv kubocker-master-*** dotfiles
$ cd dotfiles
$ source setup.sh
```

- edit dotfiles directory at $kbk_dir
```bash
$ cd dotfiles
$ source setup.sh
```
